# .gitignore Guide

**Updated**: 2026-01-16  
**Purpose**: Exclude all files that can be regenerated from source code

---

## ğŸ“‹ What's Ignored

### 1. Python Virtual Environments
```
.venv/
venv/
env/
ENV/
```

**Why**: Virtual environments are environment-specific and can be recreated.

**Regenerate**:
```bash
python -m venv .venv
source .venv/bin/activate
uv pip install -r requirements.txt
```

---

### 2. Python Build Artifacts
```
*.egg-info/
*.egg
build/
*.whl
*.pyz
*.pyzw
```

**Why**: Generated during package building.

**Regenerate**:
```bash
cd bma_send_whatsapp
source ../.venv/bin/activate
shiv -c get_chats_sync -o get_chats_sync.pyz -e get_chats_sync:main .
```

---

### 3. PyInstaller Artifacts
```
*.spec
build/
dist/
```

**Why**: Generated by PyInstaller (not currently used, but included for completeness).

---

### 4. Shiv Cache
```
.shiv/
```

**Why**: Created in user home directory when running .pyz files. Automatically regenerated.

---

### 5. Log Files
```
*.log
sync.log
bordero_status.log
```

**Why**: Generated during program execution. Should not be version controlled.

---

### 6. Test Coverage
```
.coverage
htmlcov/
.pytest_cache/
```

**Why**: Generated by test runners. Can be regenerated by running tests.

---

### 7. IDE Files
```
.vscode/
.idea/
```

**Why**: IDE-specific settings. User preferences, not project requirements.

---

### 8. OS Files
```
.DS_Store
Thumbs.db
```

**Why**: OS-specific metadata. Automatically generated.

---

### 9. Temporary Files
```
*.tmp
*.temp
*.swp
*.swo
*~
```

**Why**: Editor and system temporary files.

---

## âœ… What to Commit

**Source Code** (COMMIT):
- `*.py` - Python source files
- `*.md` - Documentation
- `setup.py` - Package definition
- `requirements*.txt` - Dependencies
- `*.json` - Configuration templates
- `.gitignore` - This file

**Generated Files** (IGNORE):
- `*.pyz` - Built binaries
- `*.egg-info/` - Build metadata
- `__pycache__/` - Python cache
- `.venv/` - Virtual environment
- `*.log` - Log files

## ğŸ”„ Regeneration Guide

All ignored files can be regenerated:

1. **Virtual Environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   uv pip install -r requirements.txt
   ```

2. **Binary**:
   ```bash
   cd bma_send_whatsapp
   source ../.venv/bin/activate
   shiv -c get_chats_sync -o get_chats_sync.pyz -e get_chats_sync:main .
   ```

3. **Other Files**: Automatically generated during build/execution
